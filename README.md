# README

Syntax highlight for free format input.

## Features

![screen capture](https://github.com/kchoworks/free-format-input/blob/main/img/screen-capture.png)

- line comment: "#"

- keyword: "\*"

- key: "=" in key=value

## Requirements

There is no requirement.

## Extension Settings

Currently, no additional setting is provided.
